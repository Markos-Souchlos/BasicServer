##To push on github run on terminal:
    1. touch .gitignore -> nano .gitignore -> node_modules + package-lock.json
    2. git init
    3. git add .
    4. git commit -m "first commit"
    5. gh repo create 
    6. git remote add origin "https://github.com/Markos-Souchlos/YourProjectName.git"
    7. git branch -M main
